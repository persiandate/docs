#Options

Name type default description
format string 'mm/dd/yyyy' the date format, combination of d, dd, m, mm, yy, yyy.
formatter Function -- format value of input
{...
formatter: function(unixDate){return unixDate}
...}
altField selector string false An input element that is to be updated with the selected date from the datepicker. Use the altFormat option to change the format of the date within this field. Leave as blank for no alternate field. acceptable value: : '#elementId','.element-class'
altFormat String unix The dateFormat to be used for the altField option. acceptable value: unix,gregorian,g,u,YYYY/mMM/DD
altFieldFormatter Function -- format value of 'altField' input
{...
altFieldFormatter: function(unixDate){return unixDate}
...}
observer Boolean false if you use $(element).val() to set input val and 'observer : true' , datepicker sync with your inputed Value. Observer Watch input and altField Elemets.
onShow Function -- A function that takes current datepicker instance. It is called just before the datepicker is displayed.
onHide Function -- A function that takes current datepicker instance. It is called just before the datepicker Hide.
onSelect Function -- A function that takes current datepicker unixDate. It is called When Day Select.
viewMode string day day,month,year
persianDigit Boolean true if true all digit convert to persian digit.
position Array auto [x,y] , define a position of datepicker relative to input element.
autoClose Boolean false If true picker close When Select day
minDate unix -- unix
maxDate unix -- unix
timePicker Object -- timePicker config object
enabled 		Boolean false Enable or Disable dayPicker
scrollEnabled 		Boolean true change view date by scroll.
secondStep 		number 1
minuteStep 		number 1
hourStep 		number 1
showSeconds 		Boolean true if 'false' second view of timepicker dos not show
showMeridian 		Boolean true if 'false' meridian view of timepicker dos not show
navigator Object -- navigator config object
enabled 		Boolean -- Enable or Disable dayPicker
text 		Object -- navigator text object
btnNextText 		String "<" text of navigator next button
btnPrevText 		String ">" text of navigator previews button
onNext 		Function -- Trigger When Next button clicked
onPrev 		Function -- Trigger When Prev button clicked
onSwitch 		Function -- Trigger When Switch button clicked
toolbox Object -- toolbox config object
enabled 		Boolean true Enable or Disable dayPicker
text 		Object -- toolbox text object
btnToday 		String "امروز" Text of today button
onToday 		Function -- Trigger When Today Button Clicked
dayPicker Object -- dayPicker config object
scrollEnabled 		Boolean true change view date by scroll.
enabled 		Boolean true Enable or Disable dayPicker
titleFormat 		String 'YYYY MMMM' format text of navigator title
titleFormatter 		Function -- format text of navigator title
onSelect 		Function -- Trigger When Date Select
monthPicker Object -- monthPicker config Object
scrollEnabled 		Boolean true change view date by scroll.
enabled 		Boolean true Enable or Disable monthPicker
titleFormat 		String 'YYYY MMMM' format text of navigator title
titleFormatter 		Function -- format text of navigator title
onSelect 		Function -- Trigger When Month Select
yearPicker Object -- yearPicker config Object
scrollEnabled 		Boolean true change view date by scroll.
enabled 		Boolean true Enable or Disable yearPicker
titleFormat 		String 'YYYY MMMM' format text of navigator title
titleFormatter 		Function -- format text of navigator title
onSelect 		Function -- Trigger When Year Select
onlyTimePicker Boolean false if True just show timepicker and other view disabled.
justSelectOnDate Boolean true if True Value of input just change by click on day in day view
checkDate Function - By using this method you make disable custom dates.
checkMonth Function - By using this method you make disable custom Month.
checkYear Function - By using this method you make disable custom Year.
