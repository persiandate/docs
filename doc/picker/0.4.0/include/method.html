#Methods

Name Arguments Description Sample Code
show This method does not accept any arguments. Open the date picker. 
$( ".selector" ).pDatepicker( "show" );
hide This method does not accept any arguments. Close the date picker. 
$( ".selector" ).pDatepicker( "hide" );
destroy This method does not accept any arguments. Removes the datepicker functionality completely. 
$( ".selector" ).pDatepicker( "destroy" );
setDate Persian Date Array, .Net Date format and ... Set datepicker selected date time. 
$( ".selector" ).pDatepicker("setDate",[1392,12,1,11,11] );
